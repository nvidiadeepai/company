Page Loading (Pre-Loader)

Learn how to change or remove pre-loading animation.

You can change page pre-loader by replacing below img path with your new pre-loader image path.

<!-- Pre loader -->
<div class="preloader">
	<img class="preloader-item" src="assets/images/loader/02.gif" alt="Loading">
</div>

Note: Page Pre-loader are enabled by default. If you wish to disable the page pre-loader you can simply delete above code from all HTML files.

We have provided 7 types of pre-loader options to choose from. Simply copy below code and paste it immediately after body tag in all HTML pages.
